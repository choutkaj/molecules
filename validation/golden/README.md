# Golden data

Store normalized JSON expectations here.

Suggested layout:

```text
validation/golden/small/<feature-id>/*.json
validation/golden/macro/<feature-id>/*.json
```

Golden files must be generated by the declared reference software from externally supplied fixtures
listed in `validation/features/<feature-id>/<corpus-id>.toml`. Store them under
`validation/features/<feature-id>/golden/<corpus-id>/`. Do not commit hand-authored molecular
expectations as reference goldens.
