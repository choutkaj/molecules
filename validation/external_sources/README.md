# External Validation Sources

This directory contains compact externally supplied molecule records used as validation fixtures.

Rules:

- Do not hand-author molecule records for reference-golden validation.
- Every file used by a feature/corpus manifest must have a matching `fixture_sources` entry with source, HTTPS URL, and SHA-256.
- Golden data derived from these files must be generated by the declared reference software, currently RDKit for small molecules and Biopython for macromolecular structure records.
- If a malformed parser input is needed, keep it in a focused unit test rather than as a golden-validation molecule fixture.
